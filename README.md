# Katsikat-Backend (Nodejs, expressjs, sequalize)
This is a demo for node.je Express (ejs) with mysql 8.0

## Clone

```sh
git clone https://github.com/qweersq/backend-katsikat.git
```

## Install

```sh
npm install
```

## Start Mysql

Using mysql

```sh
yarn db
```
## Start server

```sh
yarn start
npm start
```
